# 🎵 OST Mood Playlist

게임 OST(FFXIV, 스타듀밸리 등)를 오디오 특징 기반으로 자동 태깅하고,  
현재 기분·상황(코딩, 휴식, 집중 등)을 입력하면 맞춤 플레이리스트를 생성합니다.

## 주요 기능

- 🎼 librosa 기반 BPM·에너지·감성 자동 추출
- 🏷️ 무드·상황별 자동 태깅 (코딩, 휴식, 집중, 전투, 감성 등)
- 🔍 임베딩 기반 유사 곡 추천
- 🎮 FFXIV, 스타듀밸리, 기타 게임 OST 지원

## 기술 스택

- **Backend**: Python, FastAPI
- **오디오 분석**: librosa
- **임베딩/ML**: sentence-transformers, scikit-learn
- **저장소**: SQLite
- **Frontend**: HTML/CSS/JS (바닐라)
